Cookies and local storage

Last updated: 16 June 2026

Current use

The Markellos Ecosystem application code does not currently set advertising cookies or analytics cookies. It uses browser local storage to remember site preferences.

Values stored locally

The site may store the following values in your browser:

  • Consent preference: remembers whether you selected necessary-only storage or accepted all storage, so the preference banner does not appear on every visit.
  • Interface settings: remembers choices such as colour theme, text size, title layout, circle size, spacing, font, background detail and motion preference.
  • Circle layout: stores saved circle positions when the layout-management functionality is used.

Optional features

No optional tracking feature is currently active. Selecting “Accept” records that preference but does not currently enable advertising, analytics or other tracking technologies. If this changes, this page and the preference controls will be updated before optional technologies are enabled.

Managing stored preferences

You can remove the stored values by clearing the site data for Markellos Ecosystem in your browser. This restores the default settings and may cause the preference banner to appear again.

External websites

Each linked blog, application or website may use its own cookies and local storage. Please consult the information provided by the relevant external website.
